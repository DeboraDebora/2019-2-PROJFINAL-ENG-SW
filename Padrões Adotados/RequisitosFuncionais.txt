RF001- O sistema deve permitir o cadastro de usuários.
RF002- O sistema deve permitir consultar os usuários cadastrados.
RF003- O sistema deve permitir alterar as informações dos usuários cadastrados.
RF004- O sistema deve permitir excluir usuários registrados.
RF005- O sistema deve permitir o cadastro de contratos.
RF006- O sistema deve permitir consultar os contratos cadastrados.
RF007- O sistema deve permitir alterar as informações dos contratos cadastrados.
RF008- O sistema deve permitir excluir contratos registrados.
RF009- O sistema deve permitir o cadastro de feedback.
RF010- O sistema deve permitir consultar os feebacks cadastrados.
RF011- O sistema deve permitir alterar os feedbacks.
RF012- O sistema deve permitir excluir os feedbacks.
